¿Que es un script?

Tema #1

Los scripts son fragmentos de código que tienen como objetivo realizar o añadir funciones dentro de una página web o e-commerce. La evolución de los scripts los han convertido en parte fundamental de la programación web y sin ellos no sería posible realizar muchas de las funciones que las páginas actuales ofrecen a los usuarios.
