Comandos basicos y sus utilidades

Tema #3

1. termux-setup-storage : Este comando es para darle permiso al almacenamiento a termux

2. apt update && apt upgrade -y : Este comando sirve para actualizar los paquetes de termux

3. cd : Ir a un directorio

4. pwd : Ver en que ruta te encuentras

5. rm : Borrar un archivo/carpeta

6. ls : Lista de los archivos o carpetas

7. clear : Limpiar la terminal

8.mkdir : crear carpetas

9. touch : Crear archivos

10. cd .. : Ir a la parte interior de un directorio
